# FATE-LLM: FedCoT
FedCoT (Federated Chain-of-Thought Distillation for Large Language Models) is a novel framework for privacy-preserving federated distillation of large language models. This implementation is integrated into the FATE-LLM framework.

For more details, please refer to the paper: ["FedCoT: Federated Chain-of-Thought Distillation for Large Language Models"](https://arxiv.org/pdf/2406.12403)
